# Collector
Collector是一个简单的图片浏览，收藏，下载APP，整体功能简单朴素，可以分频道浏览网友分享的图片素材，并提供下载的功能。<br>

本项目集成了作者其他的开源组件 [Doodle](https://github.com/BillyWei01/Doodle) 以及 [FastKV](https://github.com/BillyWei01/FastKV)。
